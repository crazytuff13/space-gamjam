# Space Astronaut Game 

A 2D space platformer built in Unity with C# developed 
during a game jam. Play as an astronaut navigating 
through space levels!

##  Play the Game
[Click here to play on itch.io!](https://crazytuff.itch.io/space)

## Features
-  Main menu to start the game
-  2 unique space levels
-  Pause menu with controls guide
-  Restart level functionality
-  Background soundtrack
-  Key collectibles
-  Portal level transitions

## Controls
| Action | Key |
|--------|-----|
| Move Left/Right | Arrow Keys or A/D |
| Jump | Space |
| Pause | Escape |
| Restart Level | R |

## Scripts
- **PlayerController** - Handles player movement and actions
- **CameraFollow2D** - Smooth camera that follows the player
- **MyGameManager** - Controls overall game state
- **PauseManager** - Handles pausing, restarting and quitting
- **MusicManager** - Controls background soundtrack
- **UIAudioManager** - Manages UI sound effects
- **MainMenu** - Main menu functionality
- **Portal & PortalTrigger** - Level transition system
- **KeyTile** - Collectible key mechanics
- **QuickInputs** - Input management

## Technologies Used
- Unity Game Engine
- C#
- itch.io deployment
